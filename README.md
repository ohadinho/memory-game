# Memory Game

This is a small memory game which uses [react-boilerplate](https://github.com/react-boilerplate/react-boilerplate).

## Quick start
1. Make sure that you have Node.js v8.15.1 and npm v5 or above installed.
2. Clone this repo using `git clone --depth=1 https://github.com/ohadinho/memory-game.git <YOUR_PROJECT_NAME>`
3. Move to the appropriate directory: `cd <YOUR_PROJECT_NAME>`.
4. Run  `npm start`  and navigate to  `http://localhost:3000` on your browser.
